* Type of Booleans
The type of Booleans is inductively defined by two terms that we call "true" and "false".

#+begin_src agda2
data Bool : Type lzero where
  true : Bool
  false : Bool

{-# BUILTIN BOOL Bool #-}
{-# BUILTIN TRUE true #-}
{-# BUILTIN FALSE false #-}
#+end_src

We also define the "swap" function, which permutes the values of "true" and "false".
#+begin_src agda2
swap : Bool â†’ Bool
swap true = false
swap false = true
