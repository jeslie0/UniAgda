* Coproduct Types
The coproduct of two types is defined inductively by elements from each type.
#+begin_src agda2
data _+_ {i j : Level} (A : Type i) (B : Type j) : Type (i ⊔ j) where
  inl : A → A + B
  inr : B → A + B
#+end_src
