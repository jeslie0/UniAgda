* Functions
Agda introduces most of the required function types already. This means we don't need to define \(\Pi\) types or \(\to\) types, just their basic operations.
#+begin_src agda2
_o_ : ∀ {i₁ i₂ i₃} {A : Type i₁} {B : Type i₂} {C : Type i₃}
      (g : B → C) (f : A → B)
      → A → C
(g o f) a = g (f a)
infixr 9 _o_

id : ∀ {i} {A : Type i}
     → A → A
id a = a
#+end_src
